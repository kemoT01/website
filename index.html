Optimization project - portfolio optimization using Simulated Annealing

As part of the subject Non-classical Optimization Methods, I worked on a project involving the use of Simulated Annealing to find the optimal portfolio. I was responsible for writing approximately half of the Python code used, utilizing the SimAnneal library, and creating the necessary graphs.

Machine Learning project - Spaceship Titanic prediction competition

Technologies used - Python, pandas, SAS Enterprise Guide, SAS Enterprise Miner

As part of the subject Data Mining, I used Neural Networks in SAS Enterprise Miner software to solve a popular Kaggle Prediction Competition, Spaceship Titanic.

My work was based on pre-processing and cleaning performed in Python as a team. Afterwards, each person used different Machine Learning methods and different software to predict whether the given passenger will be "transported". Despite high accuracy offered by my final model we decided to choose one based on a Decision Tree, due to the simpler interpretation and only slightly degraded performance.
